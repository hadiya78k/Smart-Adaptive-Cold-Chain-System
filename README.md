# Smart Adaptive Cold Chain System

## Overview
The Smart Adaptive Cold Chain System is an IoT-based solution designed to maintain the required temperature for vaccines and other temperature-sensitive products.
The system continuously monitors temperature using DHT22 sensors and automatically activates a Peltier cooling module whenever the temperature exceeds the safe range.

## Features
- Real-time temperature monitoring
- Automatic cooling
- ESP32 Microcontroller
- GSM alert system
- Blynk Dashboard
- Android App
- Low power consumption

## Hardware Components
